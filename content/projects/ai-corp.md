## 概述

AI-Corp 是启元智能的旗舰产品——一个全栈 AI 驱动的企业管理平台。

## 核心功能

- 智能项目管理：AI 辅助任务拆分、风险评估、进度追踪
- 团队协作：多角色 AI Agent 协同工作
- 数据分析：实时仪表盘、自动报告生成
- 知识管理：组织知识沉淀、经验复用

## 成果与影响

- 从 V1 Demo 演进到 V7.1 可生产版本
- 验证了多角色 AI Agent 协同工作的可行性
- 为启元智能的 PDCA 流水线提供了实战验证场

## 技术亮点

前后端分离架构，前端 Next.js + React，后端 Python FastAPI，深度集成 OpenAI API 实现智能决策辅助。

## 迭代历程

经历了 V1 到 V7 七个大版本，从简单 Demo 演进为可生产使用的企业级平台。
